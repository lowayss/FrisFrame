# FrisFrame 소개 영상 제작 패키지

90초 유튜브 소개 영상용 초안입니다. 화면 녹화 소스가 준비되면 아래 순서대로 편집합니다.

## 영상 방향

FrisFrame은 콘티를 예쁘게 그리는 도구가 아니라, 촬영 전에 카메라·배우·소품의 위치와 움직임을 확인하는 프리비즈 작업 공간으로 소개합니다.

## 필요한 화면 녹화

1. 새 프로젝트 생성
2. 2D 무대에서 배우와 카메라 배치
3. 카메라 방향과 화각 변경
4. 3D 무대에서 공간 확인
5. 카메라와 배우 키프레임을 직접 추가
6. 키프레임 사이 이동을 재생
7. 멀티카메라를 추가하고 A/B/C 앵글 비교
8. 프리뷰를 확인하고 이미지 또는 MP4 내보내기

## 편집 톤

- 16:9, 90초, 빠르지만 지나치게 광고처럼 과장하지 않기
- 실제 앱 화면을 중심으로 사용 장면을 보여주기
- UI 설명은 짧은 한글 자막으로만 처리
- 배경음악은 낮게, 마우스 클릭과 재생 시작음은 작게 살리기
- 최종 화면에는 `FrisFrame · 촬영 전에 움직임을 확인하다`를 남기기
